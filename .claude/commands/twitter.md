Create an engaging tweet about what was built in this session.

Format: $ARGUMENTS

If no format provided, ask:

**Format?**
- `single` - One punchy tweet (< 280 chars)
- `thread` - 3-5 connected tweets
- `hot-take` - Opinionated single tweet
- `build-log` - "Today I built X" style

**Vibe?** (1-5)
- 1: Informative, helpful
- 2: Casual, conversational
- 3: Excited, sharing a win
- 4: Opinionated, spicy
- 5: Controversial, debate-starter

## Guidelines

- Confident but not arrogant
- Specific, not vague
- Punchy sentences
- Keep singles under 280 characters
- 3-5 hashtags max (#ClaudeAI #DesignSystems #ReactJS #FrontendDev #WebDev)
- 0-2 emojis max

## Output

Provide 1-2 options with:
- Character count (for singles)
- The hook strategy used
