Border Guidance

Not all borders should become alpha borders.

border-default vs border-default-alpha

Use border-default

Inputs, cards, switches, selects, and any edge that must read clearly.

border-border-default

Use border-default-alpha

Soft separators, translucent layers, and decorative dividers.

border-border-default-alpha

Use muted

Passive low-emphasis surfaces: skeletons, keycaps, empty icon wells.
