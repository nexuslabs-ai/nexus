Design system

CTAs inside nav use base primary-*, not a nav-specific token.

1nav-background — the whole sidebar fill
2nav-foreground — "Nexus" title + item labels
3nav-muted-foreground — "Workspace" header, counts, footer
4nav-item-hover — "Projects" (shown hovered)
5nav-item-active — "Design system" (selected)
6nav-border — divider + sidebar right edge
Note the dark-mode overlap: nav-border and nav-item-active both resolve to slate.800 (#1e293b). Intentional — the selected item reads as "pressed into" the chrome, sitting flush with the divider depth rather than raised above it.
