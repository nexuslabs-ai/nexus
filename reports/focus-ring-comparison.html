Focus-ring policy comparison

Each option below shows the actual rendered focus state under both light and dark themes, using the real OKLCH values from packages/tailwind/. The button itself uses the brand primary color in each mode.

A

Opacity modifier on brand primary

ring-primary-background/50 · status quo

Dark mode

Look: faded dark ring in light, faded white ring in dark. Soft and quiet. Couples focus to brand color.

requires re-tuning · blocked on PR A
