Role tokens used in this guide · current mode shown
Role tokenWhat it controlsCurrent value
--control-h-{sm,md,lg}Height of buttons, inputs, selects, badges
--control-padding-x-{sm,md,lg}Horizontal padding inside controls
--control-padding-y-{sm,md,lg}Vertical padding inside controls
--control-gapIcon-to-label gap inside controls
--container-pInterior padding for Cards, Dialogs, Alerts
--container-gapGap between sub-elements inside a container
--layout-section-gapVertical gap between page sections
--layout-stack-gapDefault vertical gap in Stack utilities
