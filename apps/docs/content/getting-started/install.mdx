# Install

Nexus ships as a small set of workspace packages: the React components, the
Tailwind theme they're styled with, and the design tokens underneath. Add the
packages you consume directly, plus the form and toast peers required by the
React entrypoint.

## Add the packages

For components and the Tailwind theme:

```bash
pnpm add @nexus_ds/react @nexus_ds/tailwind react-hook-form sonner
```

`@nexus_ds/react` provides the components; `@nexus_ds/tailwind` provides the
`nx:`-prefixed utility theme they're built on. `react-hook-form` and `sonner`
back the exported Form and Toaster components. Tokens come along transitively.

### Add Appearance theming

To use the runtime Appearance system (`@nexus_ds/react/appearance`), also add the
framework-agnostic engine:

```bash
pnpm add @nexus_ds/core
```

`@nexus_ds/core` owns the appearance model, theme derivation, and the first-paint
bootstrap. It is required when you use `@nexus_ds/react/appearance`. See **Theme
setup** for the full wiring.

## Wire the styles

Import the theme and the prebuilt component styles once, at your app's CSS
entry point:

```css
@import '@nexus_ds/tailwind';
@import '@nexus_ds/react/styles.css';
```

The second import matters: Tailwind doesn't scan `node_modules`, so the
utilities used _inside_ the compiled components ship as prebuilt CSS.

## Declare native color scheme

Add this to your app shell so the browser can theme native controls and the
initial page canvas before JavaScript loads:

```html
<meta name="color-scheme" content="light dark" />
```

`@nexus_ds/tailwind` also emits the matching CSS policy: `:root` advertises
light/dark support, `.dark` pins native UI to dark, and light mode stays pinned
when `.dark` is absent.

## Render a component

Every utility is prefixed with `nx:` and every color is a semantic token, so
components theme themselves. The buttons below are live `@nexus_ds/react`
components rendered straight from this Markdown page:

<div className="nx:flex nx:flex-wrap nx:gap-3 nx:my-6">
  <Button>Primary</Button>
  <Button variant="secondary">Secondary</Button>
  <Button variant="outline">Outline</Button>
  <Button variant="ghost">Ghost</Button>
</div>

Swap the base, brand, or density in the theme picker (bottom-right) and they
re-theme live — no rebuild.
